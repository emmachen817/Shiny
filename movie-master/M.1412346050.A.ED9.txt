: 因為很愛厲陰宅所以非常期待安娜貝爾
: 昨天上映第一天馬上就衝去看
: 失望的點是
: 1.
: 鋪陳的東西太多太平凡(例如前半段剛開始或是普通的對話部分有點多)
: 而且恐怖的點太少太普通
: 以至於很多時候會有點坐不住怎麼都不進入到重點
: 或是一個恐怖的點出現後沒有延續讓人情緒又下降
: 整個片子對我來講兩個讓我有出現緊張情緒的就是縫紉機瓦斯爐爆炸那邊和電梯開門
: 我鬼片從來不看完預告片
: 所以其中看到小女孩奔向門那個梗覺得讚賞覺得有創意
: 但是依然沒被嚇到...
: 2.
: 娃娃從頭到尾都沒有自己動阿~~~~~~(除了坐在搖搖椅上搖來搖去非常勉強算是)
: 不然幾乎大部分都是女鬼或是惡魔拿著控制娃娃
: 或是娃娃被移動的鏡頭非一鏡到底自己動
: 但是當你看到有東西在控制娃娃那娃娃本身就不再那麼恐怖啦
: 3.
: 有太多你期待會發生的事情或是嚇人梗或是娃娃特寫自己動通通都沒發生
: 像我一直期待照片最後洗出來會出現什麼異狀好像沒有?!!!
: (前幾篇有板友發現相機橫拍怎麼照片出來是直的算是BUG嗎?!)
: 總之就是本來很期待安娜貝爾帶給我的恐懼幾乎都沒有讓人很無感
: 看網路上講的真實故事自行腦補可能都還更恐怖
: 我個人感覺厲陰宅有8分的話安娜貝爾大概就5分
: 我甚至認為我第一次看厲陰宅前五分鐘的片段心裡對安娜貝爾娃娃產生的恐懼
: 都大於電影"安娜貝爾"
: 就是說昨天看完電影晚上抱著她睡覺也沒關係XDDD
: 所以大家一起再來複習厲陰宅吧~!!!
: 如果是對於鬼片的入門者其實這部片可能會感到還算普通

: 但是我還是很愛安娜貝爾娃娃本身所以只給"微"負雷啦~!!
